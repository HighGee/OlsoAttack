# Olso Attack
A Simple DoS Tool with support to Proxy and Socks:
  * GET flood (Direct)
  * GET flood (Proxy - Socks)

![](https://i.imgur.com/XzHJYch.png)

# Requirements
 * Python 3.7
 * aiohttp
 * aiosocks
